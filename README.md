# WisePenView

基于 React + Ant Design 的前端项目。

## 快速开始

### 1 安装前置环境（Node.js）

[Node.js 官方下载（中文）](https://nodejs.org/zh-cn/download)

### 2 项目初始化

1. 全局安装 pnpm：
   - `npm install -g pnpm`
2. 安装依赖：
   - `pnpm install`

说明：

- 若安装失败，请检查 Node.js 与 npm 是否正确安装。

### 3 启动本地测试

- Mock 模式：
  - `pnpm mock`
- 开发模式：
  - `pnpm dev`

## 常用命令

- `pnpm dev`：启动开发服务器
- `pnpm mock`：以 mock 模式启动
- `pnpm build`：构建产物
- `pnpm lint`：执行 ESLint
